
# Twitter Stream Metrics

A set of tools for collecting metrics from Twitter's APIs.

## Requirements

- Java 1.8
- Python - tested with Python 2.7

## Installing & Building

1. Install dependencies:
```shell
pip install -U textblob
python -m textblob.download_corpora
```
2. Create a jar:
```
mvn package
```
